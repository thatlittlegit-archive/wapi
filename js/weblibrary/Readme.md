# WebLibrary
Loads libraries in an array or string from a predefined object.

## Usage
```js
loadLibraries(["jquery", "mootools"]); // loads jQuery and MooTools
loadLibraries(["jquery"]);             // loads jQuery
loadLibraries("jquery");               // loads jQuery
```
